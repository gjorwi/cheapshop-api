# Base de Datos Plana

Esta carpeta contiene los archivos JSON que actúan como base de datos de la aplicación:

- `productos.json` - Almacena todos los productos con sus detalles
- `usuarios.json` - Almacena los usuarios registrados (clientes y admin)
- `pedidos.json` - Almacena los pedidos realizados

Los archivos se crean automáticamente cuando se inicia el servidor por primera vez.
